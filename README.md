# Ask Me

The Agent / MCP tool for [blog.aotoki.me](https://blog.aotoki.me) which allows you to ask questions and get answers from the blog content.

## Development

```bash
pnpm install
pnpm run dev
```

## Deployment

```bash
pnpm run deploy
```

## Type Generation

Generate TypeScript types based on your Worker configuration:

```bash
pnpm run cf-typegen
```
